# Amazon Sales Dashboard

## 1. Project Introduction

Amazon Sales Dashboard is a Power BI project created to analyze product sales, orders, units, reviews, and sales performance across different cities, states, and products.

## 2. Project Objective

* To analyze Amazon product sales performance
* To track total sales, units, and seller count
* To identify top-selling states and cities
* To analyze product-wise sales and reviews
* To create an interactive Power BI dashboard

## 3. Tools Used

* Power BI
* Power Query
* DAX
* Excel / CSV Dataset
* Data Visualization

## 8. Key Insights

1. Maharashtra has the highest sales contribution.
2. Karnataka and Tamil Nadu are also strong sales regions.
3. Product-wise analysis helps identify best-selling items.
4. Reviews help understand product popularity.
5. Date-based charts show sales and units trends over time.
